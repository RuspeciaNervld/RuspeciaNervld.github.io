相机渲染参数
Depth 深度： 值小的先渲染
Clear Flags： 选择的内容表示仅保留

  • 默认SkyBox，即清除天空盒以外的所有
  • Depth Only 仅清除深度缓冲区，不清除颜色缓冲区，比如A摄像机先渲染，B摄像机设置了清除深度但是保留A的颜色，就会将B摄像机的内容直接叠加到A上
  • Don’t Clear（不清除）：这种模式下，相机不清除任何颜色或深度缓存。结果是，每帧绘制在下一帧之上，造成涂片效果。

渲染管线

分为三大块
CPU应用程序端 -> GPU渲染管线 -> Frame Buffer

Unity内置渲染管线

CPU应用程序

剔除，排序，打包数据，绘制调用

  • 剔除分为 视椎体剔除（四棱锥求交），层级剔除（Layer），遮挡剔除（被静态物体遮挡的物体很容易剔除，但是不剔除被动态物体遮挡的）

  • 排序， 材质可以设置 render queue 的值，越小越先被渲染。 不透明物体默认2000，半透明物体默认3000，不透明队列(<2500) 按摄像机距离从近到远，半透明队列(>2500) 摄像机距离从远到近

  • 打包数据

  • 绘制调用 Draw Call

GPU 渲染管线

  • 顶点Shader

    • 将顶点坐标转化到裁剪空间（任务）
    • 视椎体变化为2ω x 2ω x 1ω的立方体(CVV立方体)
    • 可以理解为只要把裁剪空间的深度1维度拍扁(通过硬件阶段)就得到2D图像了
      可以合起来成为MVP矩阵

  • 硬件操作阶段（图元装配及光栅化阶段）

    • 裁剪操作实际上发生在裁剪空间(CVV立方体)，裁剪空间是一个方便裁剪的空间，和之前的剔除阶段难度不一样
    • 背面剔除通过判断三角形顶点是不是逆时针顺时针来判断
    • 光栅化的过程是先将顶点进行插值绘线，然后再逐行扫描补全
    • 光栅化之后每个片元拥有的数据：其中深度值肯定有，UV坐标应该也是这个时候算好的

  • 片元Shader

    • 纹理技术
      • 小纹理映射到大区域，用纹理过滤器，如双线性，三线性插值
      • 大纹理映射到小区域，用Mipmap预先生成一些，而不是直接池化
      • 纹理寻址模式
      • 纹理压缩格式
    • 光照计算
      • 直接光照+间接光照
      • 直接光照的计算是可以每一帧进行的
      • Phong = Diffuse + Specular + ambient(环境光,Phong认为给个纯色即可,但是有更多方法可以模拟效果 IBL镜面/球形光SH模拟漫反射环境光)
      • Shader效果拆解框架
        • 可以扩充SSS次表面散射

  • 输出合并

    • 处理遮挡关系，处理半透明混合（任务）
    • 整体过程 = Alpha测试 模板测试 深度测试 混合
    • 帧缓冲区(Frame Buffer) 是关键 分为 Color Depth Stencil(模板) 这几种 buffer
      • 深度测试
        • 最开始深度都是1
        • 离摄像机越近越接近0
        • 可以通过 两个选项 来控制深度测试
          • ZWrite 深度写入： 关闭之后颜色正常但是不会留下深度值
          • ZTest 深度测试：可以自定义通过深度测试的条件
        • 可以在片元着色器之前进行提前深度测试 Early-Z
      • 混合 Blending
        • 从后到前
        • 一般要关闭 ZWrite
        • 两种混合模式 alpha混合 和 柔和叠加(变亮)
      • 模板测试

        开启并且手动设置哪些像素可以通过呈现的规则

剔除设置

  • 关闭剔除 Cull Off
  • 背面剔除 Cull Back
  • 正面剔除 Cull Front

半透明渲染问题

  • 注意subshader中加入 Tags { “Queue”=”Transparent” }
  • 注意对应Pass中关闭深度写入： ZWrite Off
  • 设置混合模式 Blend SrcAlpha OneMinusSrcAlpha

URP如果需要使用双通道，则：

  • Pass1加入：Tags { “LightMode”=”SRPDefaultUnlit” }
  • Pass2加入：Tags { “LightMode”=”UniversalForward” }

贴图模式设置

  • 如果贴图突然哪里颜色不对，改成Ramp就不会超过采样边界了会直接截断

常用变量计算

  • o.pos_world = mul(unity_ObjectToWorld, v.vertex).xyz;
  • o.normal_world = normalize(mul(float4(v.normal,0.0), unity_WorldToObject).xyz);
  • o.view_dir = normalize(_WorldSpaceCameraPos.xyz - o.pos_world);
  • half NdotV = saturate(dot(normal_world, view_dir));
  • half fresnel = 1.0 - NdotV;
