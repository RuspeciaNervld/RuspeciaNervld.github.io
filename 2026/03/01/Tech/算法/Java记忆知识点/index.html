不同类型的最大值

在 Java 中，每种基本类型都有对应的包装类，其中定义了表示最大值和最小量的常量。

类型 最大值（科学计数法） 包装类常量
int 2.147e9 Integer.MAX_VALUE
long 9.22e18 Long.MAX_VALUE
float 3.4028235e38 Float.MAX_VALUE
double 1.7976931348623157e308 Double.MAX_VALUE
char 65535 (0xFFFF) Character.MAX_VALUE
short 32767 Short.MAX_VALUE
byte 127 Byte.MAX_VALUE

此外，无符号类型在 Java 中并不原生支持，但可以通过 Integer.toUnsignedString() 等方法处理。

优先队列 PriorityQueue

Java 的 PriorityQueue 默认是小顶堆（自然顺序），可以通过构造函数传入 Comparator 来改变顺序。

Vector（一般不使用）

Java 中 Vector 是遗留类，线程安全但性能较差。通常使用 ArrayList。

ArrayList 常用操作：

  • 初始化：new ArrayList<>()、new ArrayList<>(initialCapacity)、Arrays.asList(...) 返回固定大小的 List，可再传给 ArrayList 构造。
  • 添加：add(E)、add(index, E)、addAll(Collection)
  • 删除：remove(index)、remove(Object)、clear()
  • 获取/修改：get(index)、set(index, E)
  • 大小：size()
  • 遍历：for 循环、增强 for、迭代器、forEach
  • 排序：Collections.sort(list) 或 list.sort(Comparator)

输入输出格式化

Java 提供 System.out.printf 和 String.format 进行格式化输出，格式说明符与 C 类似。

常用格式符：

  • %d：整数
  • %f：浮点数
  • %s：字符串
  • %c：字符
  • %b：布尔值
  • %n：换行符（平台独立）

读取特定格式的输入可以使用 Scanner 的各种 nextXxx() 方法。
